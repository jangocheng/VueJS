# 项目介绍
该项目是一个简单电商订货平台,实现的功能有:轮播图.商品列表.商品大图预览.发表评论.个人中心.加入购物车等功能

# 技术栈
* Vue 2.0
* vue-router
* vuex
* Axios
* vue-preview
* vue-material

# 运行及构建
``` bash
# install dependencies
npm install

# serve with hot reload at localhost:9747
npm run dev

# build for production with minification
npm run build
```
